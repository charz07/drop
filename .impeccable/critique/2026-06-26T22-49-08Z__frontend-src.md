### Priority Issues

**[P1] Silent profile fetch failure** — Profile.jsx:15 swallows all errors. Users who liked brands see empty state with no way to distinguish error from genuinely empty. Fix: catch error, show retry.

**[P1] Zero help at value-loop moment** — Nothing communicates that reactions improve future drops. The product's core mechanic is invisible to new users. Fix: one line of context on Drop screen.

**[P2] Mobile layout untested** — brand-image-wrap at fixed 220px min-width breaks on 375px viewport. Reaction buttons below 44px touch target minimum. Fix: responsive column layout below 480px.

**[P2] Two shape systems** — 999px pills (quiz, reactions) vs 8/10/12px rectangles (buttons, cards). Fix: establish one radius vocabulary (4/8/12px) and apply consistently.

**[P3] side-stripe and tag contrast** — .taste-summary border-left is banned side-stripe pattern. Brand tags #888 on #222 fail WCAG AA (~3:1). Fix: full border + background tint; bump tag color to #aaa.
